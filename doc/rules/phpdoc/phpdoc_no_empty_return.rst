===============================
Rule ``phpdoc_no_empty_return``
===============================

``@return void`` and ``@return null`` annotations should be omitted from PHPDoc.

Examples
--------

Example #1
~~~~~~~~~~

.. code-block:: diff

   --- Original
   +++ New
    <?php
    /**
   - * @return null
    */
    function foo() {}

Example #2
~~~~~~~~~~

.. code-block:: diff

   --- Original
   +++ New
    <?php
    /**
   - * @return void
    */
    function foo() {}
