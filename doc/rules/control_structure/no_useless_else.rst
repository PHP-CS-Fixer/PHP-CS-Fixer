========================
Rule ``no_useless_else``
========================

There should not be useless ``else`` cases.

Examples
--------

Example #1
~~~~~~~~~~

.. code-block:: diff

   --- Original
   +++ New
    <?php
    if ($a) {
        return 1;
   -} else {
   +}  
        return 2;
   -}
   +
