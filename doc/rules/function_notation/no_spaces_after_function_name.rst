======================================
Rule ``no_spaces_after_function_name``
======================================

When making a method or function call, there MUST NOT be a space between the
method or function name and the opening parenthesis.

Examples
--------

Example #1
~~~~~~~~~~

.. code-block:: diff

   --- Original
   +++ New
    <?php
   -strlen ('Hello World!');
   -foo (test (3));
   -exit  (1);
   -$func ();
   +strlen('Hello World!');
   +foo(test(3));
   +exit(1);
   +$func();
