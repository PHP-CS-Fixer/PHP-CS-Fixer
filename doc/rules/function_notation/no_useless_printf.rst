==========================
Rule ``no_useless_printf``
==========================

There must be no ``printf`` calls with only the first argument.

Warning
-------

Using this rule is risky
~~~~~~~~~~~~~~~~~~~~~~~~

Risky when the ``printf`` function is overridden.

Examples
--------

Example #1
~~~~~~~~~~

.. code-block:: diff

   --- Original
   +++ New
    <?php

   -printf('bar');
   +print 'bar';
