==========================
Rule ``strict_comparison``
==========================

Comparisons should be strict.

.. warning:: Using this rule is risky.

   Changing comparisons to strict might change code behavior.

Examples
--------

Example #1
~~~~~~~~~~

.. code-block:: diff

   --- Original
   +++ New
    <?php
   -$a = 1== $b;
   +$a = 1=== $b;

---------

@PhpCsFixer:risky
