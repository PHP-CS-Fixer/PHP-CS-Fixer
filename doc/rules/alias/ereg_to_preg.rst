=====================
Rule ``ereg_to_preg``
=====================

Replace deprecated ``ereg`` regular expression functions with ``preg``.

Warning
-------

Using this rule is risky
~~~~~~~~~~~~~~~~~~~~~~~~

Risky if the ``ereg`` function is overridden.

Examples
--------

Example #1
~~~~~~~~~~

.. code-block:: diff

   --- Original
   +++ New
   -<?php $x = ereg('[A-Z]');
   +<?php $x = preg_match('/[A-Z]/D');
