=============================
Rule ``protected_to_private``
=============================

Converts ``protected`` variables and methods to ``private`` where possible.

Examples
--------

Example #1
~~~~~~~~~~

.. code-block:: diff

   --- Original
   +++ New
    <?php
    final class Sample
    {
   -    protected $a;
   +    private $a;

   -    protected function test()
   +    private function test()
        {
        }
    }
