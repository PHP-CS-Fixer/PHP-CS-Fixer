==========================
Rule ``heredoc_to_nowdoc``
==========================

Convert ``heredoc`` to ``nowdoc`` where possible.

Examples
--------

Example #1
~~~~~~~~~~

.. code-block:: diff

   --- Original
   +++ New
   -<?php $a = <<<"TEST"
   +<?php $a = <<<'TEST'
    Foo
    TEST;
