===============================
Rule ``no_trailing_whitespace``
===============================

Remove trailing whitespace at the end of non-blank lines.

Examples
--------

Example #1
~~~~~~~~~~

.. code-block:: diff

   --- Original
   +++ New
    <?php
   -$a = 1;     
   +$a = 1;
